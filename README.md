# Ascii Bullet Journal

> Documenting the ideas to convert a bullet journal to a digital copy.  This will help with searching and archiving

## Legend

* [.] - task incomplete
* [/] - waiting for something
* [X] - completed a task
* [-] - note
* [O] - event
